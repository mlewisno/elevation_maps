# IDEA-001: Christmas Ornament Output Mode

Small-format output (4-6 thin layers) sized for Christmas ornaments or
display pieces. No frame — island/feature centered in a compact form factor.

**Example use case**: Kaua'i island ornament, ~75mm wide, 4 layers of
1.5mm cardstock or thin plywood, with a hanging hole at the top.

**What it would need**:
- Reduced layer count mode (4-6 layers)
- Small output dimensions (~50-100mm)
- Optional hanging hole cutout
- Possibly different simplification threshold (coarser for small size)
- Could omit labels at this scale
