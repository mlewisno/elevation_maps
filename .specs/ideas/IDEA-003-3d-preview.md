# IDEA-003: 3D Preview Rendering

Generate a visual preview of how the stacked layers will look before
committing to cutting material. Options:

- **Stacked 2D SVG**: Render all layers with transparency/offset — simplest
- **matplotlib 3D**: Quick 3D surface plot showing layer steps
- **WebGL viewer**: Interactive 3D in browser — most impressive, most work
